# Ardalo Digital Platform Frontpage Service
Provides the frontpage of the public part of the Ardalo Digital Platform.

## Tech Stack

* Java 11
* Gradle
* Spring Boot
* Spring MockMvc
* Spock
* Thymeleaf

## Quick Start

### Build application

```bash
./gradlew clean build
```

### Start application

```bash
./gradlew bootRun
```
